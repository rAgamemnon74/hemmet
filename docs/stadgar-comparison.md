## HSB/Riksbyggen-specifika regler (organisationsberoende)

Dessa regler är unika för föreningar anslutna till HSB eller Riksbyggen och påverkar plattformens design:

| Regel | HSB | Riksbyggen |
|---|---|---|
| **Reserverad styrelsepost** | 1 ledamot + 1 suppleant | 1 ledamot (suppleant borttagen 2023) |
| **Reserverad revisor** | HSB Riksförbund utser 1 | Nej |
| **Veto vid stadgeändring** | HSB-styrelsens godkännande krävs | Riksbyggens godkännande krävs |
| **Obligatoriskt medlemskap** | Förvärvare måste vara HSB-medlem | Medlem i Riksbyggen intresseförening |
| **Utträdesbeslut** | 2 stämmor + 2/3 majoritet | Samma |
| **Stämmopunkter** | Val av distriktsrepresentant till HSB | — |
| **Likvidation/fusion** | Kräver HSB + HSB Riksförbunds godkännande | Riksbyggens godkännande |

**Plattformsbehov:** Konfigurerbar flagga `organizationAffiliation: 'NONE' | 'HSB' | 'RIKSBYGGEN' | 'SBC'` som aktiverar/döljer organisationsspecifika fält och stämmopunkter.

## Riksbyggen 2019→2023: viktigaste ändringarna

| Område | Förändring |
|---|---|
| **Ombud** | Kretsbegränsning **borttagen** — valfritt ombud |
| **Överlåtelseavgift** | Höjd 2,5% → **3,5%** av PBB |
| **Pantsättningsavgift** | Höjd 1% → **1,5%** av PBB |
| **Riksbyggen-suppleant** | Borttagen (bara 1 ledamot nu) |
| **Digital kallelse** | Nu tillåtet (e-post, digital anslagstavla) |
| **Digital stämma** | Nytt — helt/delvis digital stämma tillåten |
| **Underhållsansvar** | Kraftigt utökat för BRH (elledningar, persienner, golvvärme) |
| **Lägenhetsförändring** | Nya tillståndskrav (ventilation, eldstad, kulturvärden) |
| **Ny förverkandegrund** | Otillåtna ombyggnader utan tillstånd |
| **Årsredovisning** | Ska inkludera kassaflödesanalys |
| **Näringsförbud** | Person med näringsförbud kan inte sitta i styrelse |

## Konsoliderade plattformskrav baserat på 10 stadgar

### Konfigurerbara parametrar (BrfRules)

```
Föreningsanslutning:
  organizationAffiliation: NONE | HSB | RIKSBYGGEN | SBC
  reservedBoardSeats: number (0-2)
  reservedAuditorSeats: number (0-1)
  requireOrgApprovalForStatuteChange: boolean

Styrelse:
  minBoardMembers: 3         (alltid 3)
  maxBoardMembers: 7-11      (varierar)
  maxSubstitutes: 3-6        (varierar)
  allowExternalMembers: 0-2  (varierar)

Stämma:
  noticePeriodMinWeeks: 2    (alltid 2)
  noticePeriodMaxWeeks: 4-6  (oftast 6, Insteget: 4)
  noticeMethod: PHYSICAL | DIGITAL | BOTH
  allowDigitalMeeting: boolean (nytt 2023)

Ombud:
  maxProxiesPerPerson: 1-2   (oftast 1, Becknabergha: 2, Insteget: obegränsat)
  proxyCircleRestriction: boolean (äldre: ja, 2023-standard: nej)
  proxyMaxValidityMonths: 12 (alltid)

Motioner:
  motionDeadline: konfigurerbart datum/formel

Dagordning:
  agendaTemplate: per-förening konfigurerbar
  adjustersCount: 1-2
  separateVoteCounters: boolean (HSB: ja)

Avgifter:
  transferFeeMaxPercent: 2.5 | 3.5    (äldre vs 2023)
  pledgeFeeMaxPercent: 1.0 | 1.5      (äldre vs 2023)
  subletFeeMaxPercent: 10             (alltid)
  transferFeePaidBy: BUYER | SELLER   (varierar!)

Revisorer:
  minAuditors: 1-2
  maxAuditors: 2-3
  maxAuditorSubstitutes: 1-2
  requireAuthorizedAuditor: boolean

Underhåll:
  maintenancePlanRequired: boolean
  maintenanceFundPercent: number (0.3% eller enligt plan)
  maintenancePlanYears: 50      (HSB 2023)

Röstning:
  blankVoteCountsAsAbstained: boolean
  tieBreaker: CHAIRPERSON | LOTTERY (beslut vs val)
  secretBallotOnRequest: boolean (personval)

Beslutförhet:
  quorumRule: HALF_PLUS_ONE (alltid)
  notFullQuorumRule: ONE_THIRD (alltid)
```
